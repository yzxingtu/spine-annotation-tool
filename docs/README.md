# 整图椎弓根标注使用教程

本教程介绍 Spine Annotator 中**整图椎弓根标注**功能的操作方法。该功能用于在完整 X 光脊柱影像上，为每节椎骨标注左右椎弓根点位，并可导出为 Crop 训练数据集。

## 基本操作流程

### 1. 打开整图椎弓根标注窗口

从主工具界面菜单栏进入「整图椎弓根标注」，窗口将自动加载当前数据集的所有图片。

### 2. 选择椎骨

- **点击 AABB 框**：左键单击影像中任意椎骨的绿色 AABB 框，即可选中该椎骨（高亮显示黄色边框）。
- **上下箭头键**：按 `↑` / `↓` 在椎骨列表中顺序切换。
- **右侧椎骨列表**：直接点击列表中的椎骨名称。

### 3. 添加椎弓根点位

在 AABB 框范围内**双击鼠标左键**即可添加当前侧（L 或 R）的椎弓根点位。

- 若当前侧已有点位，双击时会自动切换到另一侧（L↔R）再添加。
- 添加后可立即**拖动**调整位置。

### 4. 选中与拖动点位

- **单击点位**：左键点击任意已有点位，即可选中并直接拖动到新位置。
- 选中后，点位外围会出现**黄色虚线圆圈**（选中环），表示当前正在操作的点位。

### 5. 切换侧边（L / R）

- 按 `L` 键 或 `←` 方向键：切换到图像左侧（L 侧）
- 按 `R` 键 或 `→` 方向键：切换到图像右侧（R 侧）

切换侧边后，黄色选中环会跳转到对应侧的点位（若已标注）。

### 6. 删除点位

选中某侧点位后，按 `Delete` 键或点击右侧「移除当前侧点位」按钮，可删除该点位。

### 7. 标记难点

按 `M` 键或点击右侧「标记难点」按钮，可为当前椎骨打上难点标记，方便后续复查。已标记的椎骨在列表中会显示 ⚑ 前缀。

## 键盘快捷键一览

| 快捷键 | 功能 |
|--------|------|
| `↑` / `↓` | 在椎骨列表中切换上/下一节椎骨 |
| `←` / `L` | 切换到图像左侧（L 侧） |
| `→` / `R` | 切换到图像右侧（R 侧） |
| `Ctrl+←` / `Ctrl+→` | 切换到上/下一张图片 |
| `Delete` | 删除当前选中侧的点位 |
| `M` | 标记/取消当前椎骨为难点 |
| `1` `2` `3` | 设置当前点位可见性（见下方说明） |
| `Ctrl+S` | 保存当前标注数据到缓存 |
| 鼠标滚轮 | 缩放影像 |
| 鼠标中键拖动 | 平移画布 |

## 鼠标操作说明

| 操作 | 功能 |
|------|------|
| 左键单击 AABB 框 | 选中该椎骨（高亮黄色边框） |
| 左键单击已有点位 | 选中该点位并可直接拖动 |
| 左键双击 AABB 框内空白处 | 在当前侧添加新点位 |
| 左键拖动点位 | 移动点位位置 |
| 左键单击空白区域 | 取消侧边选中（黄色环消失） |
| 右键单击 | 标记当前椎骨为难点（同 `M` 键） |
| 滚轮滚动 | 缩放影像 |
| 中键按住拖动 | 平移画布 |

## 可见性设置

每个点位可设置 3 级可见性，用于描述椎弓根在影像中的可见程度：

| 等级 | 名称 | 说明 |
|------|------|------|
| `3` | 清晰可见 | 椎弓根清晰可辨（默认值） |
| `2` | 模糊可见 | 椎弓根存在但边界模糊 |
| `1` | 遮挡 | 椎弓根被其他结构遮挡，以虚线圆显示 |

> 注：无需标注不可见的椎弓根，直接删除该点位即可。

快捷键 `1`~`3` 可直接设置当前选中点位的可见性。

## 图层控制

右侧控制面板提供图层显示开关，可按椎骨类别隐藏/显示对应的 AABB 框和点位：

- 颈椎 C（C7）
- 胸椎 T（T1–T12）
- 腰椎 L（L1–L5）
- 骶椎 S（S1）

取消勾选后，对应类别的框和点位将在画布上隐藏，但不影响数据，重新勾选即可恢复。

## 保存与自动保存

### 手动保存（Ctrl+S）

按 `Ctrl+S` 将当前所有椎弓根标注数据保存到缓存文件（`.annotate_progress.json`，位于数据集根目录）。保存成功后，状态栏显示「已保存」，图片列表中该图片名称将**置灰**，表示已保存。

### 自动保存

以下场景会自动保存当前图片的修改：
- 切换到另一张图片时（`Ctrl+←` / `Ctrl+→` 或点击列表）
- 关闭窗口时

### 图片列表颜色说明

| 颜色 | 含义 |
|------|------|
| 橙色 | 当前图片有未保存的修改 |
| 灰色（置灰） | 数据已保存到磁盘 |
| 默认色 | 尚未进行标注操作 |

## 导出 Crop 数据集

完成标注后，点击右侧「**导出 Crop 数据集…**」按钮，选择输出目录即可生成训练用 Crop 数据集。

### 导出内容

输出目录结构：
```
pedicle_crop_dataset/
├── images/        # 每张椎骨裁剪图（以 AABB 范围 +15% padding 裁剪）
├── labels/        # YOLO 格式标签文件（归一化坐标）
└── meta/          # 每张裁剪图的元数据 JSON
```

### 导出规则

- 仅导出**至少有一侧点位**（L 或 R）且可见性 > 0 的椎骨。
- 两侧点位均已删除的椎骨**不会导出**（不生成空白标签文件）。
- 椎骨按解剖学顺序处理：C7 → T1–T12 → L1–L5 → S1。
- 文件名格式：`{图片编号}-{椎骨名称}.jpg/txt/json`（如 `000769-T4.jpg`）。

### 标签文件格式

每个 `.txt` 文件为 YOLO 格式，每行一个点位：
```
{class_id} {cx} {cy} 0 0
```
其中 `cx`、`cy` 为归一化坐标（0~1），`w=0 h=0` 表示点标注。

## 推荐标注流程

1. **打开**整图椎弓根标注窗口，图片列表自动加载。
2. **选择图片**：点击图片列表或使用 `Ctrl+←/→` 切换。
3. **从 C7 开始**：点击 C7 的 AABB 框，或使用 `↓` 键逐节向下标注。
4. **添加 L 侧点位**：确认侧边为 L（按 `L` 切换），在 AABB 框内双击添加左侧椎弓根点。
5. **添加 R 侧点位**：按 `R` 切换到 R 侧，双击添加右侧椎弓根点（若 L 侧已有点位，双击会自动切换）。
6. **调整位置**：拖动点位至准确位置。
7. **切换下一节椎骨**：按 `↓` 继续标注下一节，重复步骤 4–6。
8. **标记难点**：遇到难以判断的椎骨，按 `M` 标记，后续复查。
9. **保存**：标注完一张图片后按 `Ctrl+S`，图片名称置灰表示已保存。
10. **导出**：全部完成后点击「导出 Crop 数据集…」生成训练数据。
